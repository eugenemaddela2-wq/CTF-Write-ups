# Writeup Template

## Writeup template

Use this structure for every challenge page.

### Metadata

* **Event:** `...`
* **Challenge:** `...`
* **Category:** `...`
* **Difficulty:** `...`

### Summary

Explain the goal in 2 or 3 sentences.

### Recon

List the clues, dead ends, and useful observations.

### Solution

Show the full solve path.

Include commands, payloads, and screenshots when they matter.

### Flag or takeaway

Record the flag, if you can share it.

Add the main lesson if the flag should stay private.

### Post a new writeup

1. Open the right category.
2. Create a new page named after the challenge.
3. Copy this structure and replace the placeholders.
